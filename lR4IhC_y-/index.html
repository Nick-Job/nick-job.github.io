Clash Windows客户端使用教程

2022-07-15 次阅读 2 min read

如果安装了360的软件，请立即卸载360，否则软件可能无法正常使用。Windows 10建议使用自带杀毒软件，Windows 7建议使用火绒。

Download Software — 下载软件

  • 下载对应系统版本的安装包和汉化包。
  1. Clash Windows-64位安装包：
    本站下载 （备用链接）或 官方下载
  2. Clash Windows-32位安装包：
    本站下载 （备用链接）或 官方下载
  3. 汉化包：
    本站下载 （备用链接）或 官方下载）

如果无法打开7z文件，请先安装压缩软件。

install software — 安装软件

    1. 打开安装包，选择为使用这台电脑的任何人安装，点击下一步。
    1. 点击安装，选择安装位置。
    1. 取消勾选运行Clash for Windows(R)，点击完成。
    1. 打开软件安装位置下的resources文件夹，用汉化包的app.asar替换同名文件安装语言包。
    1. 运行Clash，如果询问是否允许通过防火墙，勾选全部网络，点击允许访问。
    1. 点击右下角托盘处的Clash图标打开主界面。

Set up subscription — 设置订阅

    1. 切换到配置界面，把Clash的订阅链接复制进文本框，点击下载配置文件。
    1. 右键刚刚下载的配置，点击设置，把名称改为东方网络，把更新间隔改为24，点击确定。
